## Operation

1. Install terraform cli
2. Install & configure google cloud cli (ref. [Google Cloud CLI](https://cloud.google.com/sdk/docs/install-sdk))
3. `gcloud auth application-default login`
4. `terraform init`
5. `terraform plan`
6. `terraform apply`
